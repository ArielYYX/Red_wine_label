# Red_wine_label
1.基于Opencv库实现对红酒酒标的匹配，掌握了图像的SIFT特征提取; 2.熟悉hdf5库的分类的机制; 3.利用边缘检测确定酒标的区域并建立黑白模板; 4.利用Ransac算法对图像特征点匹配对提纯，降低图像旋转，尺寸缩放，图像平移，光照的变化的影响。 5.Qt 程序编写提取汽车模型的黑白模板，和Ransac算法匹配结果演示。
